html에 css를 스타일로 지정하는 방법
inline : html 내부에 지정

css를 inline방식으로 지정한 방식

css를 internal방식으로 지정한 방식

external 방식: css확장자를 가진 별도의 파일에 링크를 걸어주고, css는 파일를 변경하는 것을 통해 사용

선택자의 종류: 전체 선택자, 태그 요소 선택자, 아이디 선택자, 클래스 선택자

전체선택자: 별표(*)를 식별자로 사용. 모든 요소에 해당 내용 적용
태그 요소 선택자: 그냥 태그 이름(h1, a 등)
아이디 선택자: 샵(#)으로 css에서 선택해서 적용, 태그 뒤에 아이디 지정 필요(h1 id="red") 방식으로 활용
클래스 선택자: 아이디와 비슷. 점(.)으로 css에서 선택해서 적용

결합 선택자

텍스트에 적용할 수 있는 대표적인 속성: 컬러, 정렬, 줄간격, 대소문자설정, 데코레이션 등
color: darkgreen;
text-align: center;
line-height: 200%
text-decoration: line-through;

font-size: ;
font-weight: bold;
